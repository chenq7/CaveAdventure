
Cave Adventure is a survival 2D shooter game where the user controls a knight and fight off waves of monsters to win.

## Getting Started

1. Download or clone this project
2. `npm run install`
3. `npm run start`
4. Visit `localhost:1234`

## Deployment

1. `npm run build`
2. Output files are in `dist/`
